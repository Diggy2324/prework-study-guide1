HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Stands for Cascading Style Sheets
  • Describes how HTML elements are to be displayed on screen, paper, or in other media
  • Saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

Git

  • A free, open-source version control system that's used to track changes to source code over time
  • It's the most popular version control system in the world and is considered the modern standard for software developmen

Javascript

  • a programming language that adds interactivity to web pages
  • Refreshing social media feeds: JavaScript can refresh social media feeds.
  • Displaying animations: JavaScript can display animations and interactive maps.
  • Click-to-show dropdown menus: JavaScript can create click-to-show dropdown menus.
  • Dynamically changing element colors: JavaScript can change element colors on a webpage.
  • Scrolling video jukeboxes: JavaScript can create scrolling video jukeboxes.
